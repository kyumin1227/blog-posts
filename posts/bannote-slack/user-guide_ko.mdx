# Bannote Bot 사용 가이드

Bannote Bot은 Google Calendar와 연동하여 수업 시간표 구독 및 알림, 스터디룸 예약을 Slack에서 간편하게 처리할 수 있도록 도와주는 봇입니다.

# 학생 가이드

## 회원가입

GSC Slack App을 사용하려면 먼저 회원가입이 필요합니다.

#### 1단계: 홈 탭에서 가입 시작

1. Slack에서 **Bannote Bot**을 찾아 클릭합니다 ([슬랙 좌측에 앱이 보이지 않아요](#문제-해결))
2. **홈** 탭으로 이동합니다
3. **"가입하기"** 버튼을 클릭합니다

  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2015.55.58@2x.png?raw=true"
  alt="회원가입"
  max-width="100%"
  height="auto"
/>

#### 2단계: Google 계정 연동

1. 팝업 모달에서 **"Google 로그인"** 버튼을 클릭합니다
2. 브라우저가 열리면 Google 계정으로 로그인합니다
3. 권한 요청을 승인합니다 (캘린더 접근 권한 필요)

> 💡 **주의**: 가입은 학교 이메일로 제한되어 있습니다, **g.yju.ac.kr**로 끝나는 Google 계정으로 연동하세요.

#### 3단계: 가입 정보 입력

Google 로그인이 완료되면 가입 폼이 나타납니다.

| 항목      | 설명                                | 예시             |
| --------- | ----------------------------------- | ---------------- |

| 이메일    | Google 계정 이메일 (수정 불가)      | hong@g.yju.ac.kr |
| 학번/사번 | 본인의 학번 또는 사번               | 2423000          |
| 역할      | 본인에게 해당하는 역할 선택         | 키지기           |
| 반        | 소속 반 선택 (해당 시)              | 2024-A반         |

#### 4단계: 승인 대기 (역할에 따라 다름)

- **학생**: 가입 즉시 활성화됩니다 ✅
- **조교, 교수, 반대표, 키지기**: 관리자 승인 후 활성화됩니다

> 승인이 완료되면 Slack DM으로 알림을 받으며, 반을 선택 한 경우 자동으로 해당 반의 채널에 참가하게 됩니다.

## 시간표 구독하기

원하는 수업의 시간표를 구독하면 본인의 Google Calendar에 자동으로 추가됩니다.

#### 구독 방법

1. Slack 메시지 입력창에 `/구독` 명령어를 입력합니다
2. **태그 선택** 모달이 나타납니다
3. 원하는 태그를 선택합니다 (여러 개 선택 가능)
   - 예: `2024-A반`, `전공` 등
4. **"검색"** 버튼을 클릭합니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2016.40.14@2x.png?raw=true"
  alt="시간표 구독"
  max-width="100%"
  height="auto"
/>

#### 검색 결과에서 구독하기

1. 선택한 태그에 해당하는 시간표 목록이 표시됩니다
2. 각 시간표 옆의 **"구독하기"** 버튼을 클릭합니다
3. 약 5초의 딜레이 후 구독이 완료되면 버튼이 **"구독 해제"** 로 변경됩니다

> 각 시간표 옆의 **"일정 보기"** 버튼을 클릭하면 Google Calendar에서 해당 시간표의 일정을 미리 볼 수 있습니다.

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-03-17%20at%2000.48.33@2x.png?raw=true"
  alt="시간표 구독2"
  max-width="100%"
  height="auto"
/>

#### 구독 확인하기

구독한 시간표는 본인의 Google Calendar에서 확인할 수 있습니다.

1. [Google Calendar](https://calendar.google.com)에 접속합니다
2. 왼쪽 사이드바의 **"다른 캘린더"** 에 구독한 캘린더가 추가된 것을 확인합니다
3. 일정이 자동으로 표시됩니다

> Google Calendar 에서 캘린더 왼쪽의 체크 버튼을 클릭하여 캘린더 별 일정을 숨기거나 표시할 수 있습니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2016.53.56@2x.png?raw=true"
  alt="시간표 구독3"
  max-width="100%"
  height="auto"
/>

#### 구독 취소하기

1. `/구독` 명령어를 다시 입력합니다
2. 이전과 동일하게 태그를 선택하고 검색합니다
3. 현재 구독 중인 시간표는 **"구독 해제"** 버튼이 표시됩니다
4. 버튼을 클릭하여 구독을 취소합니다

> 구독을 취소하면 Google Calendar에서도 해당 캘린더가 제거됩니다.

## 스터디룸 예약

스터디룸을 예약하고 본인의 Google Calendar에서 일정을 확인할 수 있습니다.

#### 예약하기

1. Slack 메시지 입력창에 `/예약` 명령어를 입력합니다
2. 활성 상태인 스터디룸 목록이 표시됩니다
3. 원하는 스터디룸의 **"예약"** 버튼을 클릭합니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-03-17%20at%2001.03.08@2x.png?raw=true"
  alt="스터디룸 예약1"
  max-width="100%"
  height="auto"
/>

> 각 스터디룸 옆의 **"일정 보기"** 버튼을 클릭하면 Google Calendar에서 해당 스터디룸의 예약 현황을 확인할 수 있습니다.

4. 예약 정보 입력 모달이 나타납니다

| 항목      | 설명                    | 비고                         |
| --------- | ----------------------- | ---------------------------- |
| 목적      | 예약 목적 입력          | 필수                         |
| 날짜      | 예약 날짜 선택          | 필수                         |
| 시간      | 예약 시작 시간 선택     | 필수, **15분 단위**만 가능   |
| 이용 시간 | 사용할 시간 선택        | 15분 ~ 240분 (15분 단위)     |
| 참석자    | 함께 이용할 사용자 선택 | 선택, 캘린더 이벤트에 초대됨 |

5. **"예약"** 버튼을 클릭합니다
6. 예약이 완료되면 Slack DM으로 확인 메시지를 받습니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-03-17%20at%2001.05.07@2x.png?raw=true"
  alt="스터디룸 예약2"
  max-width="100%"
  height="auto"
/>

> ⚠️ **주의**: 시작 시간은 반드시 **15분 단위** (예: 09:00, 09:15, 09:30)로 입력해야 합니다. 중복된 시간대에는 예약할 수 없습니다.

3. 각 예약 항목에서 목적, 스터디룸, 날짜 및 시간을 확인할 수 있습니다

4. 내용을 변경한 후 **"수정"** 버튼을 클릭합니다

#### 예약 취소하기

1. `/내예약` 명령어를 입력합니다
2. 취소할 예약의 **"취소"** 버튼을 클릭합니다
3. 확인 모달에서 **"취소하기"** 버튼을 클릭합니다
4. Google Calendar에서 해당 이벤트가 삭제되며 Slack DM으로 확인 메시지를 받습니다

> 예약 취소는 되돌릴 수 없습니다.

# 관리자 가이드 (조교 이상)

> ⚠️ 이 섹션의 기능은 **조교(TA)** 또는 **교수** 역할을 가진 사용자만 사용할 수 있습니다.

## 관리자 명령어 요약

| 명령어          | 기능                            |
| --------------- | ------------------------------- |
| `/승인`         | 가입 대기 중인 사용자 승인/거절 |
| `/반`           | 반 목록 조회                    |
| `/반생성`       | 새로운 반 생성                  |
| `/시간표`       | 시간표 목록 조회/관리           |
| `/시간표생성`   | 새로운 시간표 생성              |
| `/반복일정생성` | 반복 수업 일정 일괄 생성        |
| `/반복일정수정` | 반복 수업 일정 일괄 수정        |
| `/반복일정삭제` | 반복 수업 일정 일괄 삭제        |
| `/스터디룸`     | 스터디룸 목록 조회/관리         |
| `/스터디룸생성` | 새로운 스터디룸 등록            |

## 사용자 승인

학생을 제외한 역할(조교, 교수, 반대표, 키지기)로 가입한 사용자는 관리자 승인이 필요합니다.

#### 승인 방법

1. Slack에서 `/승인` 명령어를 입력합니다
2. 승인 대기 중인 사용자 목록이 표시됩니다
3. 각 사용자 항목에서 다음 정보를 확인할 수 있습니다:

이름
이메일
학번/사번
요청한 역할
소속 반

4. 우측의 **메뉴(⋮)** 버튼을 클릭합니다
5. **"승인"** 또는 **"거절"**을 선택합니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2017.23.05@2x.png?raw=true"
  alt="사용자 승인"
  max-width="100%"
  height="auto"
/>

#### 승인 완료 후

승인된 사용자는 Slack DM으로 알림을 받습니다
사용자 상태가 **활성화(ACTIVE)**로 변경됩니다
이제 해당 사용자는 `/구독` 등의 명령어를 사용할 수 있습니다

## 반 관리

#### 반 목록 조회

1. `/반` 명령어를 입력합니다
2. 전체 반 목록이 표시됩니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2017.24.29@2x.png?raw=true"
  alt="반 관리"
  max-width="100%"
  height="auto"
/>

각 반에는 다음 정보가 표시됩니다:

반 이름
졸업 연도
상태 (🟢 활성 / ⚪ 졸업)

#### 반 상태 변경

- 각 반의 **상태 토글 버튼**을 클릭하여 활성/졸업 상태를 변경할 수 있습니다
- **졸업** 상태로 변경하면:
  - 신규 가입 시 해당 반을 선택할 수 없습니다
  - 기존 소속 학생 정보는 유지됩니다

#### 새로운 반 생성

1. `/반생성` 명령어를 입력합니다
2. 반 생성 폼이 나타납니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2017.26.53@2x.png?raw=true"
  alt="반 관리2"
  max-width="100%"
  height="auto"
/>

| 항목      | 설명                            | 예시     |
| --------- | ------------------------------- | -------- |
| 반 이름   | 고유한 반 이름 (입학 연도 추천) | 2024-A반 |
| 졸업 연도 | 해당 반의 졸업 연도             | 2027     |

3. **"생성"** 버튼을 클릭합니다

> 💡 반을 생성하면 동일한 이름의 **태그**와 **채널**이 자동으로 생성됩니다. 이 태그를 시간표에 연결하여 반별 시간표를 구성할 수 있습니다.

## 시간표 관리

#### 시간표 목록 조회

1. `/시간표` 명령어를 입력합니다
2. 시간표 목록이 표시됩니다
3. 상단의 **상태 필터**(활성/비활성/전체)와 **태그 필터**로 원하는 시간표를 검색할 수 있습니다
4. 목록이 많은 경우 하단의 **← / →** 버튼으로 페이지를 이동합니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-03-17%20at%2001.11.53@2x.png?raw=true"
  alt="시간표 관리"
  max-width="100%"
  height="auto"
/>

각 시간표에는 다음 정보가 표시됩니다:

과목명
연결된 태그
생성자
상태 (활성/비활성)

#### 시간표 수정/관리

각 시간표의 **"수정/관리"** 버튼을 클릭하면 수정 모달이 열립니다.

**기본 정보 수정:**

| 항목   | 설명                              | 비고 |
| ------ | --------------------------------- | ---- |
| 과목명 | 수업/과목 이름                    | 필수 |
| 설명   | 과목에 대한 간단한 설명           | 선택 |
| 태그   | 연결할 태그 선택 (복수 선택 가능) | 선택 |

**알림 채널 설정:**

시간표에 반복 일정이 생성·수정·삭제될 때 알림을 받을 Slack 채널을 지정할 수 있습니다.

- **알림 채널** 항목에서 채널을 복수 선택합니다
- 학반 태그가 연결된 경우 해당 반의 채널이 자동으로 포함됩니다

> 💡 알림 채널을 설정하지 않으면 일정 변경 알림이 발송되지 않습니다.

**편집자 관리:**

시간표의 Google Calendar를 편집할 수 있는 사용자를 관리합니다. 편집자로 등록된 사용자만 반복 일정 생성·수정·삭제 명령어를 사용할 수 있습니다.

- **"편집자 추가"** 버튼 → 사용자 선택 → Google Calendar writer 권한 부여
- 편집자 목록에서 각 편집자 옆의 **"제거"** 버튼으로 권한을 취소합니다

#### 시간표 상태 변경

각 시간표의 **상태 토글 버튼**을 클릭하여 활성/비활성 상태를 변경할 수 있습니다
**비활성** 상태로 변경하면:

- `/구독` 검색 결과에 표시되지 않습니다
- 기존 구독자의 캘린더에는 영향이 없습니다

| 태그   | 연결할 태그 선택 (복수 선택 가능) | 2026-A반, 전공                           |

3. **"생성"** 버튼을 클릭합니다

#### 시간표 생성 후

시간표를 생성하면 자동으로 **Google Calendar**가 생성됩니다.

생성 시 다음이 자동으로 처리됩니다:

- 생성자에게 **편집자 권한** 자동 부여
- 생성자가 해당 시간표를 자동으로 **구독**

이후 `/반복일정생성` 명령어를 사용하여 수업 일정을 등록하세요.

#### Google Calendar에 시간표가 보이지 않아요

1. `/구독`에서 해당 시간표가 "구독 취소" 상태인지 확인하세요
2. Google Calendar 사이드바에서 해당 캘린더가 체크되어 있는지 확인하세요
3. 브라우저를 새로고침하거나 Google Calendar 앱을 재시작해보세요

#### 가입했는데 명령어가 작동하지 않아요

학생이 아닌 역할로 가입한 경우, 관리자 승인이 필요합니다
관리자(조교/교수)에게 승인을 요청하세요

#### `/구독` 검색 결과가 없어요

선택한 태그에 연결된 시간표가 없을 수 있습니다
다른 태그 조합으로 검색해보세요
관리자에게 원하는 시간표가 생성되었는지 문의하세요
