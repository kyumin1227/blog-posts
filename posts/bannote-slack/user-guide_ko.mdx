# Bannote Bot 사용 가이드

Bannote Bot은 Google Calendar와 연동하여 수업 시간표 구독 및 알림, 스터디룸 예약을 Slack에서 간편하게 처리할 수 있도록 도와주는 봇입니다.

# 학생 가이드

## 회원가입

GSC Slack App을 사용하려면 먼저 회원가입이 필요합니다.

#### 1단계: 홈 탭에서 가입 시작

1. Slack에서 **Bannote Bot**을 찾아 클릭합니다 ([슬랙 좌측에 앱이 보이지 않아요](#문제-해결))
2. **홈** 탭으로 이동합니다
3. **"가입하기"** 버튼을 클릭합니다

/>

#### 2단계: Google 계정 연동

1. 팝업 모달에서 **"Google 로그인"** 버튼을 클릭합니다
2. 브라우저가 열리면 Google 계정으로 로그인합니다
3. 권한 요청을 승인합니다 (캘린더 접근 권한 필요)

> 💡 **주의**: 가입은 학교 이메일로 제한되어 있습니다, **g.yju.ac.kr**로 끝나는 Google 계정으로 연동하세요.

#### 4단계: 승인 대기 (역할에 따라 다름)

- **학생**: 가입 즉시 활성화됩니다 ✅
- **조교, 교수, 반대표, 키지기**: 관리자 승인 후 활성화됩니다

> 승인이 완료되면 Slack DM으로 알림을 받으며, 반을 선택 한 경우 자동으로 해당 반의 채널에 참가하게 됩니다.

## 홈 탭

가입 후 Bannote Bot의 **홈** 탭에서 모든 기능에 접근할 수 있습니다.

| 섹션                | 버튼                | 기능                                          |
| ------------------- | ------------------- | --------------------------------------------- |
| 📅 시간표           | 과목별              | 수업 시간표 구독/해제                         |
|                     | 교수별              | 교수 시간표를 Google Calendar에서 미리보기    |
|                     | 교실별              | 교실별 시간표를 Google Calendar에서 미리보기  |
|                     | 태그별              | 태그별 전체 시간표를 Google Calendar에서 보기 |
| 📋 예약             | 스터디룸            | 스터디룸 예약                                 |
|                     | 교수 상담           | 교수 상담 예약 페이지로 이동                  |
|                     | 내 예약             | 내 예약 조회/수정/취소                        |
| 링크                | ✏️ 내 정보 수정     | 학번/사번, 소속 반 수정                       |
|                     | 🗓️ 캘린더           | Google Calendar 바로가기                      |
|                     | 🔗 외부 캘린더 연동 | Google Calendar 외부 동기화 설정              |

## 시간표 구독하기

원하는 수업의 시간표를 구독하면 본인의 Google Calendar에 자동으로 추가됩니다.

#### 구독 방법

1. **홈** 탭에서 시간표 **"과목별"** 버튼을 클릭합니다
2. **태그 선택** 모달이 나타납니다
3. 원하는 태그를 선택합니다 (여러 개 선택 가능)
   - 예: `2024-A반`, `전공` 등
4. **"검색"** 버튼을 클릭합니다

  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2016.40.14@2x.png?raw=true"
  alt="시간표 구독"
  max-width="100%"
  height="auto"
/>

#### 검색 결과에서 구독하기

1. 선택한 태그에 해당하는 시간표 목록이 표시됩니다
2. 각 시간표 옆의 **"구독하기"** 버튼을 클릭합니다
3. 약 5초의 딜레이 후 구독이 완료되면 버튼이 **"구독 해제"** 로 변경됩니다

> 각 시간표 옆의 **"일정 보기"** 버튼을 클릭하면 Google Calendar에서 해당 시간표의 일정을 미리 볼 수 있습니다.

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-03-17%20at%2000.48.33@2x.png?raw=true"

/>

#### 구독 확인하기

구독한 시간표는 본인의 Google Calendar에서 확인할 수 있습니다.

1. [Google Calendar](https://calendar.google.com)에 접속합니다
2. 왼쪽 사이드바의 **"다른 캘린더"** 에 구독한 캘린더가 추가된 것을 확인합니다
3. 일정이 자동으로 표시됩니다

> Google Calendar 에서 캘린더 왼쪽의 체크 버튼을 클릭하여 캘린더 별 일정을 숨기거나 표시할 수 있습니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-02-01%20at%2016.53.56@2x.png?raw=true"
  alt="시간표 구독3"
  max-width="100%"

/>

#### 구독 취소하기

1. **홈** 탭에서 시간표 **"과목별"** 버튼을 클릭합니다
2. 이전과 동일하게 태그를 선택하고 검색합니다
3. 현재 구독 중인 시간표는 **"구독 해제"** 버튼이 표시됩니다
4. 버튼을 클릭하여 구독을 취소합니다

> 구독을 취소하면 Google Calendar에서도 해당 캘린더가 제거됩니다.

## 외부 캘린더 연동

구독한 시간표를 **애플 캘린더**, **삼성 캘린더** 등 기기 기본 캘린더 앱에서도 확인할 수 있습니다.

#### 연동 방법

1. **홈** 탭에서 **"🔗 외부 캘린더 연동"** 버튼을 클릭합니다
2. Google Calendar 공유 설정 페이지가 열립니다
3. 기기 캘린더와 동기화할 캘린더를 선택합니다
4. **"저장"** 버튼을 클릭합니다
5. 각 캘린더 앱에서 **학교 Google 계정**(g.yju.ac.kr)으로 로그인하면 일정이 자동으로 동기화됩니다

> 💡 애플 캘린더는 설정 > 앱 > 캘린더 > 계정에서, 삼성 캘린더는 앱 내 계정 추가에서 Google 계정으로 로그인하세요.

> ⚠️ **반대표·조교·교수**: 외부 캘린더 앱은 일정 **조회용**으로만 이용해주세요. 일정 수정은 반드시 **Google Calendar**에서 해주세요. 외부 앱에서 수정하면 동기화 문제가 발생할 수 있습니다.

## 교수별 시간표

**홈** 탭에서 시간표 **"교수별"** 버튼을 클릭하면 등록된 교수 목록이 표시됩니다. 각 교수 옆의 **"일정 보기"** 버튼을 클릭하면 Google Calendar에서 해당 교수의 수업 일정을 확인할 수 있습니다.

> 전체 교수 일정을 한 번에 보려면 **"전체 교수 일정 보기"** 버튼을 클릭하세요.

---

## 교실별 시간표

**홈** 탭에서 시간표 **"교실별"** 버튼을 클릭하면 등록된 교실 목록이 표시됩니다. 각 교실 옆의 **"일정 보기"** 버튼을 클릭하면 Google Calendar에서 해당 교실의 수업·예약 일정을 확인할 수 있습니다.

> 교실 시간표는 조회만 가능하며, 직접 예약할 수 없습니다.

> 💡 교실 시간표는 일정의 장소(위치) 정보를 기반으로 자동 반영됩니다. 장소가 누락되거나 오타로 인해 교실명이 별칭과 다를 경우 해당 일정이 교실 시간표에 반영되지 않을 수 있습니다.

---

## 태그별 시간표

**홈** 탭에서 시간표 **"태그별"** 버튼을 클릭하면 태그 목록이 표시됩니다. 태그 옆의 **"일정 보기"** 버튼을 클릭하면 해당 태그에 연결된 모든 시간표를 Google Calendar에서 한 번에 확인할 수 있습니다.

> 예를 들어 `2024-A반` 태그를 클릭하면 해당 반의 모든 과목 일정이 하나의 캘린더 뷰에 모여 표시됩니다.

## 시간표 알림

시간표 알림은 두 가지 경우에 발송됩니다.

#### 반복 일정 생성·수정·삭제 알림

관리자가 반복 일정을 생성·수정·삭제하면 해당 시간표에 설정된 **알림 채널**로 안내 메시지가 발송됩니다.

> 💡 알림 채널은 시간표 수정 화면에서 설정할 수 있습니다. 설정하지 않으면 알림이 발송되지 않습니다.

## 스터디룸 예약

스터디룸을 예약하고 예약 현황을 확인할 수 있습니다.

#### 예약하기

1. **홈** 탭에서 예약 **"스터디룸"** 버튼을 클릭합니다
2. 활성 상태인 스터디룸 목록이 표시됩니다
3. 원하는 스터디룸의 **"예약"** 버튼을 클릭합니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-03-17%20at%2001.03.08@2x.png?raw=true"
  alt="스터디룸 예약1"
  max-width="100%"
  height="auto"
/>

> 각 스터디룸 옆의 **"일정 보기"** 버튼을 클릭하면 Google Calendar에서 해당 스터디룸의 예약 현황을 확인할 수 있습니다.

4. 예약 정보 입력 모달이 나타납니다

| 항목      | 설명                    | 비고                         |
| --------- | ----------------------- | ---------------------------- |
| 목적      | 예약 목적 입력          | 필수                         |
| 날짜      | 예약 날짜 선택          | 필수                         |
| 시간      | 예약 시작 시간 선택     | 필수, **15분 단위**만 가능   |
| 이용 시간 | 사용할 시간 선택        | 15분 ~ 240분 (15분 단위)     |
| 참석자    | 함께 이용할 사용자 선택 | 선택, 캘린더 이벤트에 초대됨 |

> 💡 참석자를 선택하지 않으면 본인이 자동으로 참석자에 추가됩니다.

5. **"예약"** 버튼을 클릭합니다
6. 예약이 완료되면 Slack DM으로 확인 메시지를 받습니다

<img
  src="https://github.com/kyumin1227/blog-posts/blob/main/images/bannote-slack/CleanShot%202026-03-17%20at%2001.05.07@2x.png?raw=true"
  alt="스터디룸 예약2"
  max-width="100%"
  height="auto"
/>

> ⚠️ **주의**: 시작 시간은 반드시 **15분 단위** (예: 09:00, 09:15, 09:30)로 입력해야 합니다. 중복된 시간대에는 예약할 수 없습니다.

> 💡 **참고**: PC에서는 시간 입력 시 숫자를 바로 입력할 수 있습니다. 24시간 기준으로 입력 후 **Enter**를 누르면 자동으로 변환됩니다. (예: `0900` → 오전 9:00, `1815` → 오후 6:15)

## 교수 상담 예약

교수별로 등록된 Google Calendar 예약 페이지를 통해 상담 일정을 예약할 수 있습니다.

#### 예약하기

1. **홈** 탭에서 예약 **"교수 상담"** 버튼을 클릭합니다
2. 상담 예약이 가능한 교수 목록이 표시됩니다
3. 원하는 교수의 **"예약하기"** 버튼을 클릭합니다
4. 브라우저에서 해당 교수의 Google Calendar 예약 페이지가 열립니다
5. 원하는 시간대를 선택하여 예약합니다

> 💡 예약 가능한 시간대는 교수가 Google Calendar에서 직접 설정합니다. 예약 페이지가 없는 교수는 목록에 표시되지 않습니다.

## 내 예약 조회/수정/취소

스터디룸 예약과 교수 상담 예약을 한 곳에서 확인하고 관리할 수 있습니다.

#### 예약 조회하기

1. **홈** 탭에서 예약 **"내 예약"** 버튼을 클릭합니다
2. **📚 스터디룸 예약**과 **💬 교수 상담 예약** 목록이 구분되어 표시됩니다
3. 각 예약 항목에서 목적, 장소, 날짜 및 시간을 확인할 수 있습니다

#### 스터디룸 예약 수정하기

1. **홈** 탭에서 예약 **"내 예약"** 버튼을 클릭합니다
2. 수정할 예약의 **"수정"** 버튼을 클릭합니다

4. 내용을 변경한 후 **"수정"** 버튼을 클릭합니다
5. 수정이 완료되면 Slack DM으로 확인 메시지를 받습니다

> ⚠️ 참석자를 모두 제거하면 예약이 자동으로 **취소**됩니다.

학생을 제외한 역할(조교, 교수, 반대표, 키지기)로 가입한 사용자는 관리자 승인이 필요합니다.

#### 승인 방법

1. **홈** 탭에서 **"가입 승인"** 버튼을 클릭합니다

이름
이메일
학번/사번
요청한 역할
소속 반

4. 우측의 **메뉴(⋮)** 버튼을 클릭합니다
5. **"승인"** 또는 **"거절"**을 선택합니다

이제 해당 사용자는 홈 탭의 기능을 사용할 수 있습니다

이름

학번/사번
역할
반
상태

## 태그 관리

태그는 시간표를 그룹화하는 데 사용됩니다. 반을 생성할 때 자동으로 태그가 만들어지며, 필요에 따라 수동으로 추가 태그를 만들 수 있습니다.

#### 새로운 태그 생성

1. **홈** 탭에서 **"태그 생성"** 버튼을 클릭합니다
2. 태그 이름을 입력합니다 (예: `전공`, `일본어`, `특강`)
3. **"생성"** 버튼을 클릭합니다

> 💡 반 생성 시 태그가 자동으로 생성됩니다. 별도의 과목 그룹이나 특수 시간표를 만들 때 수동으로 태그를 추가하세요.

#### 태그 목록 조회

1. **홈** 탭에서 **"태그 목록"** 버튼을 클릭합니다
2. 전체 태그 목록이 표시됩니다

각 태그에는 다음 정보가 표시됩니다:

태그 이름
상태 (🟢 활성 / ⚪ 비활성)
반 자동 생성 태그 여부 (반)

#### 태그 상태 변경

- 각 태그의 **상태 토글 버튼**을 클릭하여 활성/비활성 상태를 변경할 수 있습니다
- **비활성** 상태로 변경하면 구독 검색 시 해당 태그가 표시되지 않습니다

---

## 시간표(과목) 관리

#### 새로운 시간표 생성

1. **홈** 탭에서 캘린더 관리 **"과목 생성"** 버튼을 클릭합니다
2. 시간표 생성 폼이 나타납니다

| 항목   | 설명                              | 예시                                     |
| ------ | --------------------------------- | ---------------------------------------- |
| 과목명 | 수업/과목 이름                    | 프로그래밍 기초                          |
| 설명   | 과목에 대한 간단한 설명 (선택)    | 1학년 프로그래밍 기초 과목 시간표 입니다 |
| 태그   | 연결할 태그 선택 (복수 선택 가능) | 2026-A반, 전공                           |

3. **"생성"** 버튼을 클릭합니다

시간표를 생성하면 자동으로 **Google Calendar**가 생성됩니다.

- 생성자에게 **편집자 권한** 자동 부여
- 생성자가 해당 시간표를 자동으로 **구독**

이후 홈 탭의 **반복일정 관리 > 생성** 버튼을 사용하여 수업 일정을 등록하세요.

> 💡 추가된 일정은 해당 시간표를 구독한 모든 사용자의 캘린더에 자동으로 반영됩니다.

#### 시간표 목록 조회

1. **홈** 탭에서 캘린더 관리 **"과목 목록"** 버튼을 클릭합니다
2. 시간표 목록이 표시됩니다
3. 상단의 **상태 필터**(활성/비활성/전체)와 **태그 필터**로 원하는 시간표를 검색할 수 있습니다
4. 목록이 많은 경우 하단의 **← / →** 버튼으로 페이지를 이동합니다

각 시간표에는 다음 정보가 표시됩니다:

과목명
연결된 태그
생성자
상태 (활성/비활성)

#### 시간표 수정/관리

각 시간표의 **"수정/관리"** 버튼을 클릭하면 수정 모달이 열립니다.

**기본 정보 수정:**

| 항목   | 설명                              | 비고 |
| ------ | --------------------------------- | ---- |
| 과목명 | 수업/과목 이름                    | 필수 |
| 설명   | 과목에 대한 간단한 설명           | 선택 |
| 태그   | 연결할 태그 선택 (복수 선택 가능) | 선택 |

**알림 채널 설정:**

시간표에 반복 일정이 생성·수정·삭제될 때 알림을 받을 Slack 채널을 지정할 수 있습니다.

- **알림 채널** 항목에서 채널을 복수 선택합니다
- 학반 태그가 연결된 경우 해당 반의 채널이 자동으로 포함됩니다

> 💡 알림 채널을 설정하지 않으면 일정 변경 알림이 발송되지 않습니다.

**편집자 관리:**

시간표의 Google Calendar를 편집할 수 있는 사용자를 관리합니다. 편집자로 등록된 사용자만 반복 일정 생성·수정·삭제를 할 수 있습니다.

- **"편집자 추가"** 버튼 → 사용자 선택 → Google Calendar writer 권한 부여
- 편집자 목록에서 각 편집자 옆의 **"제거"** 버튼으로 권한을 취소합니다

#### 시간표 상태 변경

각 시간표의 **상태 토글 버튼**을 클릭하여 활성/비활성 상태를 변경할 수 있습니다.

- 구독 검색 결과에 표시되지 않습니다

#### 반복 일정 삭제

1. **홈** 탭에서 **반복일정 관리 > "삭제"** 버튼을 클릭합니다
2. 삭제할 반복 일정 그룹을 선택합니다
3. 삭제 범위를 선택합니다

| 옵션      | 설명                                     |
| --------- | ---------------------------------------- |
| 전체      | 해당 반복 일정 전체 삭제                 |
| 오늘 이후 | 오늘 날짜부터 이후 일정만 삭제 (휴강 등) |

4. **"삭제"** 버튼을 클릭합니다
5. 삭제가 완료되면 알림 채널로 안내 메시지가 발송됩니다

## 리소스 관리

스터디룸, 교실, 교수 캘린더를 통합하여 **리소스**로 관리합니다. 리소스 유형에 따라 동작 방식이 다릅니다.

| 유형       | 설명                                                         |
| ---------- | ------------------------------------------------------------ |
| 스터디룸   | 사용자가 직접 예약할 수 있습니다                             |
| 교실       | 예약 불가, 시간표 일정이 자동으로 복제되어 표시됩니다        |
| 교수       | 예약 불가, 교수 일정이 미러링됩니다. 예약 페이지 URL 등록 시 상담 예약이 가능합니다 |

#### 새로운 리소스 생성

1. **홈** 탭에서 캘린더 관리 **"리소스 생성"** 버튼을 클릭합니다
2. 리소스 정보 입력 폼이 나타납니다

| 항목           | 설명                                    | 예시                  |
| -------------- | --------------------------------------- | --------------------- |
| 이름           | 공간 이름 또는 교수 이름                | 스터디룸 A / 홍길동   |
| 유형           | 스터디룸 / 교실 / 교수 중 선택          | 스터디룸              |
| 별칭           | 검색용 별칭 (쉼표로 구분)               | 301강, 301호          |
| 설명           | 부가 설명 (선택)                        | 3층 스터디룸          |
| 기본 공간      | 기본 리소스로 설정 여부                 | -                     |
| 예약 페이지 URL | 교수 유형에서만 사용, Google Calendar 예약 페이지 링크 | - |

3. **"등록"** 버튼을 클릭합니다

> 리소스를 생성하면 동일한 이름의 **Google Calendar**가 자동으로 생성됩니다.

> 💡 **교실 별칭**: 이벤트의 장소(location) 정보와 별칭이 일치해야 교실 시간표에 자동 반영됩니다.
> 💡 **교수 별칭**: 이벤트 location의 `/` 뒤 텍스트와 매핑됩니다.

#### 리소스 목록 조회

1. **홈** 탭에서 캘린더 관리 **"리소스 목록"** 버튼을 클릭합니다
2. 전체 리소스 목록이 표시됩니다

각 리소스에는 다음 정보가 표시됩니다:

이름
유형 (스터디룸 / 교실 / 교수)
상태 (🟢 활성 / ⚪ 비활성)

#### 리소스 상태 변경

각 리소스의 **상태 토글 버튼**을 클릭하여 활성/비활성 상태를 변경할 수 있습니다.

**비활성** 상태로 변경하면:

- 스터디룸: 예약 목록에 표시되지 않습니다
- 교실: 교실 시간표에 표시되지 않습니다
- 교수: 교수별 시간표 및 상담 예약 목록에 표시되지 않습니다

#### 리소스 정보 수정

1. **홈** 탭에서 캘린더 관리 **"리소스 목록"** 버튼을 클릭합니다
2. 수정할 리소스의 **"수정"** 버튼을 클릭합니다
3. 수정 모달이 나타납니다

| 항목           | 설명                                    | 비고 |
| -------------- | --------------------------------------- | ---- |
| 이름           | 공간 이름 또는 교수 이름                | 필수 |
| 유형           | 스터디룸 / 교실 / 교수                  | 필수 |
| 별칭           | 검색용 별칭 (쉼표로 구분)               | 선택 |
| 설명           | 부가 설명                               | 선택 |
| 상태           | 활성 / 비활성                           | 필수 |
| 예약 페이지 URL | 교수 유형 전용, Google Calendar 예약 페이지 링크 | 선택 |

4. 변경 후 **"저장"** 버튼을 클릭합니다

#### 수정자 관리

스터디룸 예약은 수정자(writer)로 등록된 사용자의 Google 계정을 통해 캘린더 이벤트를 생성합니다. 수정자가 없으면 예약이 불가능하므로 반드시 한 명 이상 등록해야 합니다.

1. **홈** 탭에서 캘린더 관리 **"리소스 목록"** 버튼을 클릭합니다
2. 관리할 스터디룸의 **"수정자 관리"** 버튼을 클릭합니다
3. 현재 수정자 목록이 표시됩니다

**수정자 추가:**

- Slack 사용자 목록에서 추가할 사용자를 선택합니다
- 선택한 사용자가 Google Calendar에 writer 권한으로 등록됩니다

> 💡 수정자로 등록된 사용자는 해당 스터디룸의 Google Calendar를 직접 편집할 수 있습니다.

2. **Bannote Bot**을 찾아 클릭 후 "앱 열기" 버튼을 클릭합니다

#### Google Calendar에 시간표가 보이지 않아요

1. 홈 탭 > 시간표 **과목별** 에서 해당 시간표가 "구독 취소" 상태인지 확인하세요
2. Google Calendar 사이드바에서 해당 캘린더가 체크되어 있는지 확인하세요
3. 브라우저를 새로고침하거나 Google Calendar 앱을 재시작해보세요

#### 가입했는데 기능이 작동하지 않아요

학생이 아닌 역할로 가입한 경우, 관리자 승인이 필요합니다
관리자(조교/교수)에게 승인을 요청하세요

#### 구독 검색 결과가 없어요

선택한 태그에 연결된 시간표가 없을 수 있습니다
다른 태그 조합으로 검색해보세요
관리자에게 원하는 시간표가 생성되었는지 문의하세요
